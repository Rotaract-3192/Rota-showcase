# Developer Utility Scripts

## Purpose
This directory contains reference documentation and automated scripts for backend engineering workflows. It provides a standardized approach for team members executing common tasks.

## Script References
- **`generate-types.md`**: Instructions and aliases for running the Supabase TypeScript generator.
- **`backup.md`**: Runbook for performing manual pg_dump backups and archiving them to cold storage.
- **`restore.md`**: Runbook for pointing a backup file to a local or staging database for disaster recovery drills.
- **`deploy.md`**: Guide for applying migrations and edge functions to production safely.

*To be populated with actual `.sh` or `.ps1` executable scripts in future development sprints.*
